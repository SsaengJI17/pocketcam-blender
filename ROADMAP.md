# Roadmap

## Version 0.1.0 - Rotation MVP

- Blender add-on receives pose data from local Python test sender.
- Blender camera rotation updates in real time.
- Android app streams Rotation Vector Sensor quaternion.
- USB ADB reverse connection confirmed.

## Version 0.2.0 - 6DoF MVP

- ARCore integration.
- Position + rotation streaming.
- Coordinate conversion and scale adjustment.
- Recenter calibration.

## Version 0.3.0 - Recording Tools

- Record start/stop.
- Automatic keyframe insertion.
- Keyframe baking.
- Basic smoothing.

## Version 0.4.0 - Production Features

- Take management.
- Lens and FOV controls.
- Tracking quality indicators.
- Connection diagnostics.

## Version 1.0.0 - Public Release

- Stable Android APK.
- Stable Blender add-on ZIP.
- Installation guide.
- Example project.
- Demo video.
- GitHub Actions release automation.
